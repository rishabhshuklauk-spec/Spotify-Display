Spotify Display README

What is the project?
My project is the starter spotify display project with a modification that I have turned the back side of the spotify display into a 2 inch speaker.
I have also made sure to add bluetooth connectivity to give it dual functionality as both a bluetooth speaker and a passive spotify display.

Why did I make this project?
I listen to a lot of music in my free time so seeing a spotify display felt like a no brainer choice.
I also thought it would be a good opportunity to work on my CAD skills and general electronics skills.
